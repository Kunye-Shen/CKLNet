# CKLNet
Class Knowledge-Guided Lightweight Network for Salient Object Detection of Strip Steel Surface Defect, [Kunye Shen](https://scholar.google.com.hk/citations?user=q6_PkywAAAAJ&hl=zh-CN)， [Xiaofei Zhou](https://scholar.google.com.hk/citations?user=2PUAFW8AAAAJ&hl=zh-CN)，and [Zhi Liu](https://scholar.google.com.hk/citations?user=Sd5VB2cAAAAJ&hl=zh-CN).

## Required libraries
numpy 1.18.1  
Python 3.7   
PyTorch 1.4.0  
torchvision 0.5.0  

## Usage
1. Clone this repo
```
https://github.com/Kunye-Shen/CKLNet.git
```

1. We also provide the predicted saliency maps ([GoogleDrive](https://drive.google.com/drive/folders/1u1BoysWTmzt-bilYBuyY6YC6-N4NYOUb?usp=drive_link) or [baidu](https://pan.baidu.com/s/1knrIGHuuIfhsomxLs_K-NA) extraction code: w4wd.).
